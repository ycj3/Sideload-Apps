# Sideload Apps

本仓库收录 在非越狱 iOS 设备上安装 IPA 应用的入门指南与资源合集，适合新手快速上手。
支持通过 AltStore、SideStore 等方式进行侧载，常见用途包括：微信双开、淘宝双开 等。

## 常见问题

### **Q1：SideStore 相比 AltStore 最大的优势是什么？**

**A：SideStore 只在首次安装时需要 AltServer，之后应用刷新都可以在手机本地自动完成，无需每周连接电脑。**
